# Submissions

This directory is where you store your phase submission evidence.

Create one subdirectory per phase:

```
submissions/
├── phase-1-discovery/
│   ├── discovery-report.md
│   ├── stakeholder-register.md
│   └── screenshots/
├── phase-2-landing-zone/
│   ├── hld.md
│   ├── lld.md
│   └── screenshots/
├── phase-3-networking-security/
├── phase-4-iac-automation/
├── phase-5-kubernetes-gitops/
├── phase-6-cicd-observability/
├── phase-7-database-architecture/
├── phase-8-migration-finops/
└── phase-9-arb-presentation/
```

**Important:** Push your evidence here as you complete each phase.
Your GitHub commits serve as timestamped proof of work.
