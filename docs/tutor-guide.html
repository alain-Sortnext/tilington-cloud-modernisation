Wrong-Direction Traps

TRAP 1 — CloudWatch vs Cross-AZ Cost (Phase 4 / 8)
The January cost report shows CloudWatch DataProcessing: $89.40 — this looks like a monitoring verbosity problem. The real issue is AWS Data Transfer Regional: $156.30 — this is cross-AZ traffic that can be eliminated with free VPC gateway endpoints for S3 and DynamoDB. James Patel's email hints at this but doesn't give the answer. Candidates who fix CloudWatch logs only miss the architectural problem. Strong candidates fix both: reduce log verbosity AND add VPC endpoints.
TRAP 2 — Aurora Instance Sizing (Phase 7 / 8)
SRV-006 (Oracle primary) has 32 vCPU and 256GB RAM. A naive candidate might spec db.r6g.8xlarge (32 vCPU, 256GB) to match. This is wrong. Aurora manages memory differently to a bare-metal Oracle server. The correct approach is to run AWS SCT/DMS in test, measure actual query patterns and connection counts, then right-size. db.r6g.2xlarge (8 vCPU, 64GB) is the starter placeholder — the candidate must justify with workload analysis, not server spec parity.
TRAP 3 — DR Replica = PII in eu-west-1 (Phase 7 / 8)
Aurora Global Tables or cross-region read replicas for customer data would put UK GDPR PII (customer profiles, NI numbers, account data) into eu-west-1 (Ireland). This violates UK GDPR Article 44. The AuditTrail DynamoDB table also cannot use Global Tables for the same reason — plus FCA audit records must stay in UK jurisdiction. Correct DR strategy: encrypted non-PII configuration replicated to eu-west-1; Aurora PITR snapshots only (no live PII replica); AuditTrail DynamoDB stays eu-west-2 only.
TRAP 4 — AuditTrail DynamoDB TTL (Phase 7)
DynamoDB TTL is useful for NotificationDelivery (90-day retention). But candidates often apply TTL logic to AuditTrail too. This is wrong. FCA SYSC 9.1 and COBS record-keeping requirements mean some audit records must be retained indefinitely. AuditTrail table must have NO TTL attribute. Object Lock in COMPLIANCE mode (not GOVERNANCE) on the export S3 bucket. Candidates who set TTL on AuditTrail fail the FCA compliance criterion.
